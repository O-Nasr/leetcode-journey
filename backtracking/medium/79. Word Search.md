public class Solution
{
    public bool Exist(char[][] board, string word)
    {
        int rows = board.Length;
        int cols = board[0].Length;
        for (int r = 0; r < rows; r++)
        {
            for (int c = 0; c < cols; c++)
            {
                if (DFS(r, c, 0))
                    return true;
            }
        }

        return false;

        bool DFS(int r, int c, int index)
        {
            // ✅ found full word
            if (index == word.Length)
                return true;

            // ❌ invalid
            if (r < 0 || r >= rows || c < 0 || c >= cols ||
                board[r][c] != word[index])
                return false;

            char temp = board[r][c];
            board[r][c] = '#'; // mark visited

            // explore 4 directions
            bool found =
                DFS(r + 1, c, index + 1) ||
                DFS(r - 1, c, index + 1) ||
                DFS(r, c + 1, index + 1) ||
                DFS(r, c - 1, index + 1);

            board[r][c] = temp; // 🔁 backtrack

            return found;
        }
    }
}

```csharp
public bool Exist(char[][] board, string word)
{
int rows = board.Length;
int cols = board[0].Length;

    for (int r = 0; r < rows; r++)
    {
        for (int c = 0; c < cols; c++)
        {
            if (DFS(r, c, 0)) return true;
        }
    }

    return false;

    bool DFS(int r, int c, int index)
    {
        // ❌ bounds + mismatch check FIRST
        if (r < 0 || r >= rows || c < 0 || c >= cols ||
            board[r][c] != word[index])
            return false;

        // ✅ found full word
        if (index == word.Length - 1)
            return true;

        char temp = board[r][c];
        board[r][c] = '#'; // mark visited

        int[] dr = { 0, 1, 0, -1 };
        int[] dc = { 1, 0, -1, 0 };

        for (int i = 0; i < 4; i++)
        {
            int nr = r + dr[i];
            int nc = c + dc[i];

            if (DFS(nr, nc, index + 1))
                return true;
        }

        // 🔁 undo
        board[r][c] = temp;

        return false;
    }
}

```

Ahh—this is a **very sharp observation** 👌
Both solutions are *logically equivalent*, but yes, the first one is often **faster in practice**. Let’s break down *why*.

---

# 🧠 Core Reason

> The first solution avoids **extra work per recursive call**.

Even though both explore similar trees, the **constant factors** differ.

---

# 🔍 Key Differences

---

## ⚡ 1. No loop overhead (major factor)

### First solution:

```csharp
bool found =
    DFS(r + 1, c, index + 1) ||
    DFS(r - 1, c, index + 1) ||
    DFS(r, c + 1, index + 1) ||
    DFS(r, c - 1, index + 1);
```

👉 Direct calls
👉 No loop
👉 No array indexing

---

### Second solution:

```csharp
int[] dr = { 0, 1, 0, -1 };
int[] dc = { 1, 0, -1, 0 };

for (int i = 0; i < 4; i++)
{
    int nr = r + dr[i];
    int nc = c + dc[i];

    if (DFS(nr, nc, index + 1))
        return true;
}
```

👉 Extra overhead:

* Loop control
* Array access (`dr[i]`, `dc[i]`)
* Index calculations

---

## 💡 Why this matters

Backtracking calls DFS **thousands/millions of times**.

Even tiny overhead becomes:

```text
small cost × huge number of calls = noticeable slowdown
```

---

# ⚡ 2. Short-circuiting is more direct

First version:

```csharp
DFS(...) || DFS(...) || DFS(...) || DFS(...)
```

👉 Stops immediately when one is `true`
👉 No loop checks needed

---

Second version:

```csharp
for (...)
{
    if (DFS(...))
        return true;
}
```

👉 Still short-circuits, BUT:

* Needs loop condition checks
* Needs index increment

---

# ⚡ 3. No array allocation inside DFS

Second version:

```csharp
int[] dr = { 0, 1, 0, -1 };
int[] dc = { 1, 0, -1, 0 };
```

👉 This is **VERY important**

If inside DFS:

* These arrays may be re-created each call ❌
* Causes memory overhead + GC pressure

---

First version:

```text
No allocations at all ✅
```

---

# ⚡ 4. Simpler control flow

First version is:

```text
Straight-line logic → easier for CPU optimization
```

Second version:

```text
Loop + indexing → slightly heavier
```

---

# 📊 Summary

| Factor         | First  | Second          |
| -------------- | ------ | --------------- |
| Function calls | Direct | Inside loop     |
| Loop overhead  | ❌      | ✅               |
| Array access   | ❌      | ✅               |
| Allocations    | ❌      | ⚠️ possible     |
| Performance    | Faster | Slightly slower |

---

# 🧠 Important Insight

> Both have the **same algorithmic complexity**, but different **constant factors**.

---

# 🔥 How to fix the second version

If you want it clean *and* fast:

```csharp
int[] dr = { 0, 1, 0, -1 };
int[] dc = { 1, 0, -1, 0 };
```

👉 Move these **outside DFS** (global or class-level)

---

# 🚀 Final takeaway

* First version = **optimized for speed**
* Second version = **cleaner / more scalable pattern**
