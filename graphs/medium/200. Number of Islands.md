# 🧠 Problem

Given a grid:

```text
'1' = land
'0' = water
```

👉 Count **connected components of land** (islands)

* Connections: **up, down, left, right**
* Each island = group of connected `'1'`

---

# 💡 Core Idea

> Every time you see a `'1'`, that’s a **new island** → run DFS/BFS to “sink” it.

---

# 🔁 DFS Solution (Most Common)

```csharp
public class Solution
{
    public int NumIslands(char[][] grid)
    {
        if (grid == null || grid.Length == 0)
            return 0;

        int rows = grid.Length;
        int cols = grid[0].Length;
        int count = 0;

        for (int r = 0; r < rows; r++)
        {
            for (int c = 0; c < cols; c++)
            {
                if (grid[r][c] == '1')
                {
                    count++;
                    DFS(r, c);
                }
            }
        }

        return count;

        void DFS(int r, int c)
        {
            // ❌ out of bounds or water
            if (r < 0 || r >= rows || c < 0 || c >= cols || grid[r][c] == '0')
                return;

            // mark visited (sink island)
            grid[r][c] = '0';

            DFS(r + 1, c);
            DFS(r - 1, c);
            DFS(r, c + 1);
            DFS(r, c - 1);
        }
    }
}
```

# 🌊 BFS Solution (Alternative)

```csharp
public int NumIslands(char[][] grid)
{
    int rows = grid.Length;
    int cols = grid[0].Length;
    int count = 0;

    int[] dr = { 0, 1, 0, -1 };
    int[] dc = { 1, 0, -1, 0 };

    for (int r = 0; r < rows; r++)
    {
        for (int c = 0; c < cols; c++)
        {
            if (grid[r][c] == '1')
            {
                count++;
                var queue = new Queue<(int, int)>();
                queue.Enqueue((r, c));
                grid[r][c] = '0';

                while (queue.Count > 0)
                {
                    var (cr, cc) = queue.Dequeue();

                    for (int i = 0; i < 4; i++)
                    {
                        int nr = cr + dr[i];
                        int nc = cc + dc[i];

                        if (nr >= 0 && nr < rows && nc >= 0 && nc < cols &&
                            grid[nr][nc] == '1')
                        {
                            queue.Enqueue((nr, nc));
                            grid[nr][nc] = '0';
                        }
                    }
                }
            }
        }
    }

    return count;
}
```

# 🌳 Mental Model

Grid = graph

```text
Each cell = node
Edges = adjacent cells
```

So this becomes:

> Count connected components

---

# 🔥 Key Insight

```text
See '1' → new island → flood fill it
```

---

# ⚡ Complexity

```text
Time:  O(m * n)
Space: O(m * n) worst (recursion or queue)
```

---

# ⚠️ Common Mistakes

* ❌ Forget marking visited → infinite loop
* ❌ Diagonal movement (not allowed)
* ❌ Not checking bounds first

---

# 🧩 Pattern Recognition

This is:

```text
Grid + DFS/BFS + Flood Fill
```
