[133. Clone Graph](https://leetcode.com/problems/clone-graph/description/?envType=study-plan-v2&envId=top-interview-150) (Medium)
---

# 🧠 Problem

Given a node in a graph:

```text
node.val
node.neighbors
```

👉 Create a **deep copy**:

* New nodes
* Same structure
* No shared references with original

---

# 🔥 Core Challenge

Graphs can have:

* Cycles 🔁
* Shared neighbors

👉 So you **must avoid cloning the same node multiple times**

---

# 💡 Key Idea

Use a map:

```csharp
Dictionary<Node, Node> map
```

👉 Meaning:

```text
original node → cloned node
```

---

# 🧠 Why we need it

Without map:

```text
Cycle → infinite recursion ❌
Duplicate nodes ❌
```

---

# 🧱 Node Definition

```csharp
public class Node
{
    public int val;
    public IList<Node> neighbors;

    public Node(int val)
    {
        this.val = val;
        neighbors = new List<Node>();
    }
}
```

---

# 🔁 DFS Solution (Recursive)

```csharp
public class Solution
{
    private Dictionary<Node, Node> map = new();

    public Node CloneGraph(Node node)
    {
        if (node == null)
            return null;

        if (map.ContainsKey(node))
            return map[node];

        // clone current node
        Node clone = new Node(node.val);
        map[node] = clone;

        // clone neighbors
        foreach (var neighbor in node.neighbors)
        {
            clone.neighbors.Add(CloneGraph(neighbor));
        }

        return clone;
    }
}
```

---

# 🌊 BFS Solution (Iterative)

```csharp
public Node CloneGraphBFS(Node node)
{
    if (node == null)
        return null;

    var map = new Dictionary<Node, Node>();
    var queue = new Queue<Node>();

    queue.Enqueue(node);
    map[node] = new Node(node.val);

    while (queue.Count > 0)
    {
        var current = queue.Dequeue();

        foreach (var neighbor in current.neighbors)
        {
            if (!map.ContainsKey(neighbor))
            {
                map[neighbor] = new Node(neighbor.val);
                queue.Enqueue(neighbor);
            }

            map[current].neighbors.Add(map[neighbor]);
        }
    }

    return map[node];
}
```

# 🌳 Visual Intuition

Original:

```text
1 -- 2
|    |
4 -- 3
```

Map evolves like:

```text
1 → 1'
2 → 2'
3 → 3'
4 → 4'
```

---

# ⚡ Key Pattern

```text
1. If node already cloned → return it
2. Otherwise:
   - create clone
   - store in map
   - recursively clone neighbors
```

---

# ⚠️ Common Mistakes

❌ Not using a map
→ infinite loop

❌ Cloning neighbors before saving node in map
→ cycle breaks

---

# ⏱ Complexity

```text
Time:  O(V + E)
Space: O(V)
```
