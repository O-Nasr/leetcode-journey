## Initial Thoughts

**Example:** `"(1+(4+5+2)-3)+(6+8)"`

**Strategy:** Use a stack to save/restore `result` and `sign` when entering/leaving parentheses.

**Rules:**
- Digit → build the number, add `sign * num` to result
- `+` or `-` → update sign
- `(` → push `(result, sign)` onto stack, reset both
- `)` → pop from stack, combine with current result

**Walkthrough:**

```
Initialize: result = 0, sign = 1, stack = []

'('  → push (0, 1)        stack = [(0, 1)]          result = 0, sign = 1
'1'  → result = 1
'+'  → sign = +1
'('  → push (1, +1)       stack = [(0,1), (1,+1)]   result = 0, sign = 1
'4'  → result = 4
'+'  → sign = +1
'5'  → result = 9
'+'  → sign = +1
'2'  → result = 11
')'  → pop (1, +1)        result = 1 + (+1 * 11) = 12
'-'  → sign = -1
'3'  → result = 12 + (-1 * 3) = 9
')'  → pop (0, +1)        result = 0 + (+1 * 9) = 9
'+'  → sign = +1
'('  → push (9, +1)       stack = [(9, +1)]          result = 0, sign = 1
'6'  → result = 6
'+'  → sign = +1
'8'  → result = 14
')'  → pop (9, +1)        result = 9 + (+1 * 14) = 23

Final result = 23
```

> **Note:** Numbers can be multi-digit (e.g., `"1-11"`, `"2147483647"`) — handled with inner while loop marked `(*)` in code.

public int Calculate(string s) {
    int result = 0;
    int sign = 1;
    Stack<(int, int)> stack = new Stack<(int, int)>();
    int i = 0;

    while (i < s.Length)
    {
        char c = s[i];

        if (char.IsDigit(c))
        {
            int num = 0;
            while (i < s.Length && char.IsDigit(s[i])) // (*) handles multi-digit numbers like "1-11"
            {
                num = num * 10 + (s[i] - '0');
                i++;
            }
            result += sign * num;
            continue;
        }
        else if (c == '+' || c == '-')
        {
            sign = (c == '+') ? 1 : -1;
        }
        else if (c == '(')
        {
            stack.Push((result, sign));
            result = 0;
            sign = 1;
        }
        else if (c == ')')
        {
            int prevResult = result;
            (result, sign) = stack.Pop();
            result += sign * prevResult;
        }

        i++;
    }

    return result;
}
